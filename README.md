# Esther Israel — Personal Portfolio Site

> Multi-disciplinary portfolio covering Virtual Assistance, 
> Web3 & DeFi Operations, and Frontend Development.

🔗 **Live site:** [personal-portfolio-site-ten-rouge.vercel.app](https://personal-portfolio-site-ten-rouge.vercel.app)

---

## Overview

This is my personal portfolio site — built from scratch in React, TypeScript, and Tailwind CSS. It serves three separate audiences across three dedicated pages:

- **/va** — Virtual Assistant & Operations services
- **/web3** — Web3, DeFi, and community work
- **/dev** — Frontend development projects and skills

Each section has its own color identity, layout, and content - all under one unified site.

---

## Features

- Three discipline-specific pages with unique gradient color systems
- Discipline-coded navigation — purple for VA, blue/indigo for Dev, copper/gold for Web3
- Fully responsive across mobile, tablet, and desktop
- DeFi protocol tools section covering Solana, BSC, and TON ecosystems
- Dark theme throughout with toggle option for light theme
- Smooth page transitions and consistent component architecture
